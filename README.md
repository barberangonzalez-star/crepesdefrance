# 🥞 Crepes de France — Landing Page

Landing page oficial para **Crepes de France**, Santa Cruz de la Sierra, Bolivia.

## 🚀 Desplegar en Vercel

### Opción 1: Desde GitHub (recomendado)

1. Sube esta carpeta a un repositorio en GitHub
2. Ve a [vercel.com](https://vercel.com) y conecta tu cuenta de GitHub
3. Haz clic en **"New Project"** → selecciona el repositorio
4. Vercel detecta automáticamente que es un sitio estático
5. Haz clic en **"Deploy"** → ¡listo! 🎉

### Opción 2: Vercel CLI

```bash
npm i -g vercel
cd crepes-de-france
vercel
```

## 📋 Contenido del sitio

- **Hero** con llamada a acción
- **Menú de 8 productos** con filtro por categoría (dulce / salado / especial)
- **Modal de preview** por producto con botón directo a WhatsApp
- **Sección de Eventos** completa
- **Mapa** con ubicación en Castelnau 9, Santa Cruz
- **Botones de acceso rápido**: WhatsApp, Instagram, Google Maps
- **Botón flotante** de WhatsApp en todas las páginas
